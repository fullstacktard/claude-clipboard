# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-10-26

### Added
- Initial release
- Smart clipboard paste for Claude Code
- Automatic screenshot detection and saving
- Ctrl+V interception via AutoHotkey v2
- WSL clipboard bridge
- Image counter and session tracking
- Multiple utility commands (paste-latest, paste-clear, paste-flush)
- Auto-startup script for Windows
- Comprehensive documentation

### Features
- `claude-paste-image` - Save and paste clipboard images
- `claude-paste-latest` - Re-paste the most recent image
- `claude-paste-clear` - Clear all saved images
- `claude-paste-flush` - Reset image counter
- `claude-paste-image-multi` - Multi-image support
- `claude-paste-stage` - Stage images for later use
- AutoHotkey script with Windows Terminal detection
- PowerShell clipboard watcher
- Automatic startup shortcut creation
